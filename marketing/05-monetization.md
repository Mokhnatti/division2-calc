# Этап 5: Монетизация (после 1000 DAU)

## Triggers для старта монетизации
- ✅ 1000 DAU стабильно (3+ месяца)
- ✅ Низкий bounce rate (<60%)
- ✅ 2+ минуты среднее время на сайте
- ✅ Устойчивый источник трафика (не только всплеск)

## 💰 Каналы

### 5.1. Google AdSense (приоритет HIGH)
**Статус:** Регистрация планируется на жену — тайский адрес + счёт (из памяти)

**Шаги:**
1. Подать заявку на https://adsense.google.com
2. Пройти модерацию (1-2 недели)
3. Вставить код рекламы в index.html
4. **Стратегические места:**
   - Header баннер (728x90)
   - Sidebar (300x250)
   - Between content sections (NOT disruptive)
5. Monitoring: CPM, CTR, RPM

**Ожидаемый доход:**
- 1000 DAU = 30k/month page views
- CPM ~$1-2 для игровых сайтов
- Месячный доход: $30-60 с 1000 DAU
- При 5000 DAU: $150-300/мес

### 5.2. Donation / Patreon / Бусти
**Плюс:** Нет навязчивой рекламы, лояльная база
**Минус:** Маленький % пользователей донатит

**Реализация:**
- Кнопка "☕ Поддержать проект" в footer
- Ссылка на BuyMeACoffee / Patreon / Бусти
- Объяснить куда идут деньги (хостинг, время разработки)

### 5.3. Партнёрки
**Tier 1: Игровые магазины**
- G2A / CDKeys / Kinguin — баннер "Купи Division 2 со скидкой"
- Получать комиссию с каждой покупки

**Tier 2: Geforce Now / Cloud gaming**
- Баннер "Играй в Division 2 без мощного ПК"

### 5.4. Premium фичи (future)
Опционально — если хватит аудитории:
- **Save builds** — регистрация + сохранение билдов в профиле
- **Build sharing** — публичная галерея билдов
- **Advanced filters** — сложные AND/OR/NOT комбинации
- **API access** — для других разработчиков

Модель: freemium (основа бесплатно, premium за $5/мес)

## 📊 Метрики монетизации
- Ad revenue (AdSense dashboard)
- Donation revenue (Patreon/Бусти)
- Click-through rate на аффилейты
- Paying users (если premium)

## ⚠️ Риски
- AdSense могут заблокировать за нарушения — важно читать правила
- Реклама может ухудшить UX — следить за bounce rate
- Партнёрки могут навредить репутации — выбирать только хорошие бренды

## Дедлайн
- Старт монетизации: месяц 3-4 от запуска
- Окупаемость домена + времени: ориентировочно 6 месяцев
